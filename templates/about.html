About this MLOps project

The US Visa Prediction project is an end-to-end MLOps pipeline designed to predict visa approval outcomes using machine learning. Built by Ben Gregory John, this project showcases expertise in ETL pipelines, scalable, data-driven solutions, leveraging tools like FastAPI, Pandas, Scikit-learn, Docker, AWS services, and GitHub Actions.

MLOps Pipeline Overview

Data Ingestion

Collects and loads visa application data from MongoDB Atlas database using Pandas for seamless integration.

Data Validation

Ensures data consistency, completeness, and correctness by checking for missing values and data drifts using Evidently.

Data Transformation

Prepares data through encoding, scaling, and feature engineering with NumPy and Scikit-learn.

Model Training

Uses neuromf library to find the best model and trains it with optimized hyperparameters.

Model Evaluation

Assesses model performance using accuracy, precision, recall, and F1-score metrics.

Model Pusher

Deploys the model to production using AWS with CI/CD via GitHub Actions.

Infrastructure & Deployment

AWS S3

Stores raw data, processed datasets, and model artifacts for efficient data management.

AWS ECR

Manages Docker container images for consistent and scalable application deployment.

Docker

Containerizes the application for reproducible environments across development and production.

CI/CD Pipelines

Automates testing, building, and deployment using GitHub Actions for rapid updates.
